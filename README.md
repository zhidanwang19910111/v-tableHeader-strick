# v-table-header-strick
[![code style: prettier](https://img.shields.io/badge/code_style-prettier-ff69b4.svg?style=flat-green)]

Vue directive to make table header strick

## Install

```bash
$ npm install --save v-table-header-sticker
```

```bash
$ yarn add v-table-header-sticker
```

## Use

```js
import Vue from 'vue'
import vTableHeaderStricker from 'v-table-header-sticker'

Vue.use(vTableHeaderStricker)
```

Or use it as a directive

```js
import vTableHeaderStricker from 'v-table-header-sticker'

<script>
  export default {
    directives: {
      vTableHeaderStricker: vTableHeaderStricker.directive
    }
  }
</script>
```
